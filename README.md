# 📱 LAB-23: JNI Protection & Anti-Debug (Android)

## 📖 Description

This project is an Android application that demonstrates the use of **JNI (Java Native Interface)** with basic **anti-debugging and protection techniques** in native code (C/C++).

The goal is to enhance application security by integrating native-level checks and preventing reverse engineering.

---

## 🚀 Features

* 🔗 Integration of JNI (Java ↔ C/C++)
* 🛡️ Anti-debugging techniques in native code
* ⚙️ Native function execution (e.g., factorial calculation)
* 📱 Simple and clean Android UI
